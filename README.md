# postman_API's

Collection of Postman API collections and environment files for testing and documenting APIs.

## Contents

- `collections/` - Postman collection JSON files
- `environments/` - Postman environment JSON files

This repository stores Postman collections and environments used for testing, sharing, and documenting APIs.
Import the collections into Postman to run requests, view examples, and share documentation.
